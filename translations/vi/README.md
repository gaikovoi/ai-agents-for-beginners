> **Ưu tiên Clone Cục bộ?**
>
> Kho lưu trữ này bao gồm hơn 50 bản dịch ngôn ngữ, điều này làm tăng đáng kể kích thước tải xuống. Để clone mà không có bản dịch, hãy sử dụng sparse checkout:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Điều này cung cấp cho bạn tất cả những gì bạn cần để hoàn thành khóa học với tốc độ tải xuống nhanh hơn nhiều.

## 🌱 Bắt đầu

Khóa học này có các bài học bao quát những kiến thức cơ bản về xây dựng Tác nhân AI. Mỗi bài học đề cập đến chủ đề riêng nên bạn có thể bắt đầu từ bất cứ đâu bạn thích!

Khóa học có hỗ trợ đa ngôn ngữ. Xem các [ngôn ngữ có sẵn ở đây](#-multi-language-support).

Nếu đây là lần đầu bạn xây dựng với các mô hình AI Tạo sinh, hãy xem khóa học [Generative AI For Beginners](https://aka.ms/genai-beginners), gồm 21 bài học về xây dựng với GenAI.

Đừng quên [thêm sao (🌟) cho kho lưu trữ này](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) và [fork kho lưu trữ này](https://github.com/microsoft/ai-agents-for-beginners/fork) để chạy code.

### Gặp gỡ các Học viên khác, Nhận câu trả lời cho câu hỏi của bạn

Nếu bạn gặp khó khăn hoặc có bất kỳ câu hỏi nào về xây dựng Tác nhân AI, hãy tham gia Kênh Discord riêng của chúng tôi trong [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Những gì bạn cần

Mỗi bài học trong khóa học này bao gồm các ví dụ mã, có thể tìm thấy trong thư mục code_samples. Bạn có thể [fork kho lưu trữ này](https://github.com/microsoft/ai-agents-for-beginners/fork) để tạo bản sao riêng của mình.

Các ví dụ mã trong các bài tập này sử dụng Microsoft Agent Framework với Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Cần Tài khoản Azure

Khóa học này sử dụng các framework và dịch vụ Tác nhân AI sau từ Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Để biết thêm thông tin về cách chạy mã cho khóa học này, xem phần [Cài đặt Khóa học](./00-course-setup/README.md).
