> **מעדיף לשכפל מקומית?**
>
> מאגר זה כולל למעלה מ-50 תרגומים בשפות שונות, מה שמגדיל משמעותית את גודל ההורדה. כדי לשכפל בלי תרגומים, השתמש ב-sparse checkout:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> זה נותן לך את כל מה שאתה צריך כדי להשלים את הקורס במהירות הורדה גבוהה יותר.

## 🌱 התחלת עבודה

הקורס הזה כולל שיעורים המכסים את היסודות של בניית סוכני AI. כל שיעור עוסק בנושא משלו, אז תתחיל איפה שאתה רוצה!

יש תמיכה רב-שפתית לקורס זה. עבור אל [השפות הזמינות כאן](#-multi-language-support).

אם זו הפעם הראשונה שלך בבנייה עם מודלים של AI גנרטיבי, עיין בקורס שלנו [Generative AI For Beginners](https://aka.ms/genai-beginners), הכולל 21 שיעורים על בנייה עם GenAI.

אל תשכח [להעניק כוכב (🌟) למאגר זה](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ו-[להוציא מזלג למאגר זה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי להריץ את הקוד.

### פגש לומדים אחרים, קבל מענה לשאלותיך

אם אתה תקוע או יש לך שאלות לגבי בניית סוכני AI, הצטרף לערוץ הדיסקורד הייעודי שלנו ב-[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### מה שצריך

כל שיעור בקורס זה כולל דוגמאות קוד שנמצאות בתיקיית code_samples. תוכל [להוציא מזלג למאגר זה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי ליצור עותק משלך.

דוגמאות הקוד בתרגילים אלו מנצלות את Microsoft Agent Framework עם Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - נדרש חשבון Azure

קורס זה משתמש במסגרת שירותי סוכן AI הבאים מ-Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


למידע נוסף על הרצת הקוד לקורס זה, עבור אל [הגדרת הקורס](./00-course-setup/README.md).
